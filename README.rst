==========
 workflow
==========


Workflow provides a tiny Domain Specific Language (two operators:
``|`` and ``&``) for parallel and sequential (respectively) evaluation
of Python functions.

==============
 Installation
==============

::

  pip install -r requirements.txt
  pip install .
